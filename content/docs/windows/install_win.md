  **Важно перед началом:** Перед любой установкой создайте резервные копии всех важных данных. Чистая установка полностью удалит содержимое выбранного раздела или всего диска.
{{< /callout >}}

## Где скачать официальный ISO-образ

Для установки необходим образ системы. Загрузить его можно двумя способами:

1.  **Официальный сайт Microsoft:**
    Перейдите на [microsoft.com](https://www.microsoft.com/ru-ru/software-download/windows11), выберите «Скачать образ диска Windows 11 (ISO)» → Windows 11 (multi-edition ISO) → Язык → 64-bit Download.
2.  **Каталог образов:**
    Вы можете воспользоваться моим каталогом оригинальных файлов: [Перейти к загрузке](../download_win).

> Образ должен быть формата `.iso` без модификаций, предпочтительно оригинальный (MSDN).


## Подготовка загрузочного носителя

### Способ 1: Rufus
*Рекомендуется для записи одной системы. Позволяет автоматически обойти требования TPM 2.0 и Secure Boot.*


{{% steps %}}

### Скачивание Rufus
Скачайте последнюю версию с [официального сайта](https://rufus.ie/) (портативная версия, установка не требуется).

### Подготовка накопителя
Вставьте флешку объемом **минимум 8 ГБ**.
{{< callout type="error" >}}
  Все данные на флешке будут безвозвратно удалены.
{{< /callout >}}

### Настройка записи
Запустите Rufus, выберите флешку в поле **Устройство**. Нажмите **ВЫБРАТЬ** и укажите скачанный ISO-образ Windows.

Рекомендуемые параметры для ПК 2018 года и новее:
*   **Схема разделов:** GPT
*   **Целевая система:** UEFI (non CSM)
*   **Файловая система:** NTFS

{{< callout type="info" >}}
  **Для старых ПК:** Если компьютер выпущен до 2012 года и поддерживает только Legacy BIOS, выберите схему разделов **MBR** и целевую систему **BIOS** (или UEFI-CSM).
{{< /callout >}}

### Обход ограничений
Нажмите **СТАРТ**. В открывшемся окне "Параметры пользовательского опыта":
*   Поставьте галочку **Удалить требование наличия 4 ГБ+ ОЗУ, Secure Boot и TPM 2.0** (если ПК не проходит проверку).
*   При необходимости отключите сбор данных и требование онлайн-учетной записи.

### Завершение
Подтвердите форматирование и дождитесь окончания записи.

### Способ 2: Ventoy
*Мультизагрузочная флешка. Позволяет хранить несколько ISO (Windows, Linux, утилиты) на одном носителе.*

1.  Скачайте Ventoy с [официального сайта](https://ventoy.net).
2.  Запустите `Ventoy2Disk.exe`, выберите флешку и нажмите **Install** (в опциях рекомендуется включить **Secure Boot Support**).
3.  После установки откроется раздел Ventoy. Просто скопируйте в него любые ISO-файлы.
4.  **Для обхода требований Windows 11:** Создайте в корне флешки папку `ventoy`, а внутри файл `ventoy.json` со следующим содержимым:

```json {filename="ventoy.json"}
{
    "control": [
        { "VTOY_DEFAULT_MENU_MODE": "1" },
        { "VTOY_TREE_VIEW_MENU_STYLE": "0" },
        { "VTOY_WIN11_BYPASS_CHECK": "1" }
    ]
}
```


### Способ 3: Без флешки

*Установка через временный раздел на диске. Подходит, если нет свободного USB-накопителя.*


{{% steps %}}

### Монтирование образа
Скачайте ISO и откройте его двойным кликом (подключится как виртуальный диск).

### Создание раздела
Запустите файл `setup.exe` с зажатыми клавишами {{< key "Shift+F10" />}}, чтобы открыть командную строку. Введите команды по очереди:

```powershell {filename="CMD"}
diskpart
list disk
select disk 0
create partition primary size=10000
format fs=ntfs quick label=WIN11TEMP
assign letter=W
exit
```

*Примечание: `select disk 0` — выберите диск, где есть свободное место.*

### Копирование файлов
Скопируйте файлы установщика на новый раздел (замените `E:` на букву смонтированного ISO):

```powershell {filename="CMD"}
xcopy E:\*.* W:\ /E /H /C /I /Y
```
### Загрузка
Перезагрузите ПК, зайдите в Boot Menu ({{< key >}}F8{{< /key >}}, {{< key >}}F11{{< /key >}} или {{< key >}}F12{{< /key >}}) и выберите загрузку с раздела **WIN11TEMP**.

{{% /steps %}}


## Процесс установки

1. Загрузитесь с подготовленного носителя.
2. Выберите язык и нажмите "Установить".
3. Выберите Выборочная установка (Custom Install).

## Работа с разделами диска

{{< tabs items="UEFI + GPT (Рекомендуется), Legacy BIOS + MBR (Старые ПК)" >}}
{{< tab >}}
### Для современных систем (2015+ г.в.):

1. Удалите **ВСЕ** разделы на целевом диске, чтобы осталось только «Нераспределённое пространство».
2. Нажмите "Далее".
3. Установщик автоматически создаст структуру GPT (EFI, MSR, Recovery).

{{< callout type="error" >}}
**Ошибка "Установка на данный диск невозможна (стиль GPT)":**
Если вы видите ошибку, значит вы загрузились в режиме Legacy, а диск размечен в GPT. Перезагрузитесь в BIOS и отключите CSM/Legacy Mode, оставив только UEFI.
{{< /callout >}}
{{< /tab >}}

{{< tab >}}

### Для старых систем:

1. Удалите системные разделы.
2. Создайте новый раздел и отформатируйте его в NTFS.

{{< callout type="error" >}}
**Ошибка "Установка на данный диск невозможна (стиль MBR)":**
Если ошибка возникает при попытке установить на MBR диск, значит загрузчик запущен в режиме UEFI. Вам нужно либо включить Legacy Boot в BIOS, либо конвертировать диск в GPT (через командную строку `diskpart` -> `clean` -> `convert gpt`).
{{< /callout >}}

## Финальная настройка (OOBE)

После копирования файлов и перезагрузки начнется первичная настройка.

{{< callout type="info" >}}
**Как создать локальную учетную запись:**
Чтобы не вводить данные Microsoft аккаунта, отключите интернет (выдерните кабель) на этапе настройки. Если кнопка "У меня нет интернета" отсутствует:

1. Нажмите {{< key >}}Shift + F10{{< /key >}}.
2. Введите `oobe\bypassnro` и нажмите Enter.
3. ПК перезагрузится, и опция оффлайн-установки появится.
{{< /callout >}}
